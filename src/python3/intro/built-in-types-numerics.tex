\subsection{Numeric types}
All numeric types (except complex) support:
\mintinline{python}{x + y},
\mintinline{python}{x - y},
\mintinline{python}{x * y},
\mintinline{python}{x / y} (quotient),
\mintinline{python}{x // y} (floored quotient),
\mintinline{python}{x % y},
\mintinline{python}{abs(x)},
\mintinline{python}{x ** y} (power).

The standard library includes the additional numeric types \mintinline{python}{fractions.Fraction}, for rationals, and \mintinline{python}{decimal.Decimal}, for floating-point numbers with user-definable precision.
