\textbf{git reset}\git{reset} resets staging area to last commit but leaves working directory unchanged.

{\color{red} \textbf{To be done: git reset --soft}}.

{\color{red} \textbf{To be done: git reset --mixed}}.

\mintinline{bash}{git reset --hard} resets both staging area and working directory to last commit, overwrites all changes in the working directory!

% \item [---] \texttt{hard}: resets both index and working tree.
% \item [---] \texttt{mixed}: resets index but not the working tree.
% \item [---] \texttt{soft}: doesn't touch the index file, working tree.
