\textbf{git status}\git{status} shows the working tree status.
Useful short options:
\begin{compactenum}
\item [\texttt{-b}] shows the branch even in short-format (equivalent to \mintinline{bash}{--branch}),
\item [\texttt{-s}] gives output in short format (equivalent to \mintinline{bash}{--short}).
\end{compactenum}
