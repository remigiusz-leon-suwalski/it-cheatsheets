\textbf{git tag}\git{tag} lists tags.
Useful short options:
\begin{compactenum}
\item [\texttt{-a}] makes an annotated tag instead (equivalent to \mintinline{bash}{--annotate}),
\item [\texttt{-d}] deletes existing tags instead (equivalent to \mintinline{bash}{--delete}),
\item [\texttt{-f}] instead of failing, replaces an existing tag (equivalent to \mintinline{bash}{--force}),
\item [\texttt{-m}] instead of prompting, uses given message (equivalent to \mintinline{bash}{--message=}),
\item [\texttt{-s}] makes a GPG-signed tag (equivalent to \mintinline{bash}{--sign}),
\item [\texttt{-u}] makes a custom GPG key signed tag (equivalent to \mintinline{bash}{--local-user=}),
\item [\texttt{-v}] verifies GPG signature of tags (equivalent to \mintinline{bash}{--verify}).
\end{compactenum}

\mintinline{bash}{git tag --contains <commit>} lists only tags which contain the specified commit, implies \mintinline{bash}{--list}.

\mintinline{bash}{git tag --sort=<key>} sorts based on the key given, in descending order if key is prefixed with a minus sign.
