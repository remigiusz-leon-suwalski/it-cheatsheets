\textbf{git rm}\git{rm} removes files from working tree and index.

\begin{minted}{bash}
# unstage and remove paths only from the index,
# leaving the working tree files alone
git rm --cached <path>
\end{minted}

Useful short options:
\begin{compactenum}
\item [\texttt{-f}] overrides the up-to-date check (equivalent to \mintinline{bash}{--force}),
\item [\texttt{-n}] performs a dry run instead (equivalent to \mintinline{bash}{--dry-run}),
\item [\texttt{-r}] recursively when leading directory name is given,
\end{compactenum}
